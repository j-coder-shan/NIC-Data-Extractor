# NIC Data Extractor

Milestone 1 scaffold for a production-grade Sri Lankan NIC document intelligence system.

## Current status

- FastAPI application bootstrap
- Clean architecture folder structure
- Placeholder `/api/extract` endpoint
- Baseline tests for startup wiring

## Run locally

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
uvicorn main:app --reload
```
